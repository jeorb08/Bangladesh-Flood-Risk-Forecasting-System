# Bangladesh-Flood-Risk-Forecasting-System
Bangladesh Flood Risk Forecasting System — Multi-step rainfall and river level prediction with district-level alert map

Multi-step rainfall & flood risk forecaster

A system that ingests historical rainfall, river water level, and temperature data for Bangladesh, forecasts rainfall and river levels 1–7 days ahead, classifies each district into a flood risk tier (low / medium / high / severe), and displays the results on an interactive Folium map with district-level alerts.
